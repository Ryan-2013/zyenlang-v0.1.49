# tree-sitter-zyenlang

Experimental Tree-sitter grammar source for ZyenLang v0.1.x.

For Zed dev-extension use, run from the ZyenLang project root:

```bash
python tools/install_zed_support.py
```

That script will initialize this grammar folder as a local git repository and write the correct `file:///...` grammar path into `ide/zed/zyenlang/extension.toml`.
